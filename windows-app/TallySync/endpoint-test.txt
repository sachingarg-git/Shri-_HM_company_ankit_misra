# Windows App Localhost Connection Test

## Test These URLs in Windows App:

1. **Web Connection Test**:
   ```
   GET https://your-app.replit.app/api/test-web-connection
   ```

2. **Sync Status**:
   ```
   GET https://your-app.replit.app/api/tally-sync/sync/status
   ```

3. **Heartbeat**:
   ```
   POST https://your-app.replit.app/api/tally-sync/heartbeat
   Body: {"clientId": "WINDOWS_APP_CLIENT"}
   ```

## Windows App Configuration Fix:

Change localhost URLs to actual Replit app URL:
- OLD: http://localhost:5000
- NEW: https://your-app.replit.app

The Windows app should connect to cloud server, not localhost!
